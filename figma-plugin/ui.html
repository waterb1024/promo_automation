검수용: [zip만 만들기]는 Way 호출 없이 zip만 생성.

부가서비스 템플릿
사용 방법

위치별로 라이브러리 인스턴스 1개씩 등록. 지자체마다 서비스명만 다르고 프레임 규격은 같으므로 위치당 1 템플릿 재사용.
라이브러리 인스턴스 선택 → 위치 종류 선택 → [+ 등록]

상세페이지 섹션 템플릿
사용 방법

라이브러리 컴포넌트 배치 → 선택 → 섹션 종류·이름 입력 → [+ 등록].
쿠폰처럼 여러 디자인이면 반복 등록.

설정 백업

팀원과 템플릿 설정을 공유하거나 새 기기에서 복원할 때 사용합니다.

팀 공용 저장소 (GitHub 등) 에서 최신 템플릿 설정을 받아옵니다. URL 은 helper config 의 templates_sync_url.
