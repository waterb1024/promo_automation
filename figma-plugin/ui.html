사용 방법

1. 시안 페이지에서 올릴 frame 선택
2. 위 Way 이슈 키 입력
3. [Way 업로드] — frame → PNG → zip → Way 첨부 + 댓글 자동

검수용: [zip만 만들기]는 Way 호출 없이 zip만 생성.

팝업 템플릿

라이브러리 인스턴스(팝업) 선택 → [+ 등록]
삭제: 적용 탭 dropdown 선택 후 [삭제]

상세페이지 섹션 템플릿
사용 방법

라이브러리 컴포넌트 배치 → 선택 → 섹션 종류·이름 입력 → [+ 등록].
쿠폰처럼 여러 디자인이면 반복 등록.

설정 백업
